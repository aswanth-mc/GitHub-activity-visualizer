## ✨ Features

* 👤 **Profile Overview**

  * Avatar, bio, followers, following, repositories count

* 📦 **Repositories Section**

  * Top active repositories
  * Stars ⭐ and Forks 🍴
  * Forked repo indication
  * Direct link to GitHub repo

* 💻 **Languages Used**

  * Visual representation of languages
  * Color-coded progress bars

* 📈 **Commits Over Time**

  * Line chart showing recent commit activity

* 🟩 **Contribution Heatmap**

  * GitHub-style activity grid
  * Color intensity based on commits

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox + Grid)
* JavaScript (Vanilla JS)
* GitHub REST API
* Chart.js

---

## ⚙️ How It Works

1. Enter a GitHub username
2. Fetch data from GitHub API
3. Process:

   * User profile
   * Repositories
   * Languages
   * Events (commits)
4. Display data in an interactive dashboard

## ⚠️ Limitations

* GitHub Events API provides **limited recent activity only**
* Contribution graph is a **simulated version**, not full GitHub data
